# 🏪 Store App - Flutter E-commerce

A simple e-commerce mobile application built with Flutter, featuring a robust local API architecture following enterprise-level patterns and best practices.

## 📱 Features

### Core Features

- **Product Management**: Full CRUD operations for products
- **Shopping Cart**: Complete cart management with persistent storage
- **User Authentication**: Secure login/register system
- **Order Processing**: Checkout flow with order confirmation
- **Favorites**: Personal wishlist functionality
- **Search & Filter**: Advanced search with multiple filters
- **Categories**: Hierarchical category system
